## Overview

Call to Doctor iOS app brings healthcare to your fingertips, enabling seamless video consultations between patients and healthcare providers. The app features a robust architecture using MVVP pattern for maintainable and scalable code.

## Technology Stack

### Mobile Development
- Swift 3 - Native iOS development
- MVVP Architecture - For clean separation of concerns
- CoreData - Local data persistence
- AVFoundation - Video call implementation
- HealthKit - Health data integration

### Backend Integration
- RESTful APIs - Backend communication
- WebRTC - Real-time video streaming
- Push Notifications - User alerts
- JWT - Authentication

## Key Features
- Real-time video consultations
- Secure medical data handling
- Appointment scheduling
- Digital prescriptions
- Payment integration
- Doctor profiles and ratings
- Medical history tracking
