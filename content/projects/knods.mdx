## Overview

Knods empowers freelancers with a powerful mobile platform for managing their work and clients. Built using modern technologies and following Agile practices, the application delivers a seamless user experience across devices.

## Technology Stack

### Mobile Development
- Flutter - Cross-platform framework
- Dart - Programming language
- BLoC - State management
- Firebase - Backend services

### Backend & Web
- Golang - Server runtime
- React.js - Web interface
- PostgreSQL - Database
- Docker - Containerization

## Key Features
- Project management
- Client communication
- Time tracking
- Payment integration
- Document handling
- Performance analytics
- Cross-platform support
