## Overview

The Qiscus website serves as the primary online presence for PT Qiscus Tekno Indonesia, featuring product information, technical documentation, and company updates. Currently undergoing modernization with Vue.js integration.

## Technology Stack

### Frontend
- Vue.js - Modern UI framework
- jQuery - Legacy support
- SCSS - Styling
- Webpack - Build tool

### Backend
- PHP Laravel - Web framework
- MySQL - Database
- Redis - Caching
- Nginx - Web server

## Key Features
- Product showcase
- Technical documentation
- Blog platform
- Case studies
- Contact forms
- Performance optimization
- SEO implementation
