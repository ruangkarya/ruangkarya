## Overview

The Qiscus Web SDK provides developers with powerful chat functionality while maintaining a small footprint. Built with performance in mind, it uses minimal external dependencies and modern web technologies.

## Technology Stack

### Frontend
- Vue.js - UI framework
- VanillaJS - Core functionality
- Stylus - CSS preprocessing
- Pug - HTML templating
- MQTT - Real-time messaging

### Build Tools
- Webpack - Bundling
- Babel - JavaScript compilation
- PostCSS - CSS processing
- ESLint - Code quality

## Key Features
- Real-time messaging
- File sharing
- User presence
- Chat rooms
- Message history
- Push notifications
- Custom theming
