## Overview

Kiwari Chat Web provides seamless communication through a modern web interface. Built on Vue.js with Vuex for state management, it delivers reliable and scalable chat functionality across platforms.

## Technology Stack

### Frontend
- Vue.js - JavaScript framework
- Vuex - State management
- SCSS - Styling
- ES6 - Modern JavaScript
- Qiscus SDK - Chat functionality

### Development Tools
- Webpack - Build system
- Babel - JavaScript compiler
- ESLint - Code quality
- Jest - Testing framework

## Key Features
- Real-time messaging
- File sharing
- User presence
- Message history
- Push notifications
- Group chats
- Message search
