## Overview

Kiwari Chat iOS provides seamless messaging experience for Apple device users. Built natively with Swift 3, the app leverages the Qiscus Chat SDK to deliver reliable real-time communication features.

## Technology Stack

### Mobile Development
- Swift 3 - Native iOS development
- Qiscus Chat SDK - Core chat functionality
- CoreData - Local data persistence
- Push Notifications - Real-time alerts
- AVFoundation - Media handling

### Architecture
- MVVM Pattern
- Protocol-Oriented Programming
- Dependency Injection
- Clean Architecture

## Key Features
- Real-time messaging
- Media sharing
- Push notifications
- Offline support
- User presence
- Chat history
- Group messaging
