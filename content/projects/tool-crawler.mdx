## Overview

Tool Crawler automates the collection and processing of e-commerce data, providing valuable insights for business intelligence and market analysis through efficient web scraping technology.

## Technology Stack

### Core Development
- VB.Net - Application framework
- SQL Server - Data storage
- HTML Agility Pack - Web scraping
- Regular Expressions - Pattern matching

### Features
- Multi-threading
- Proxy support
- Data parsing
- Error handling

## Key Features
- Automated data collection
- E-commerce site support
- Data processing
- Export capabilities
- Scheduling system
- Pattern recognition
- Error recovery
