## Overview

The Qiscus SDK Dashboard provides a centralized platform for managing and monitoring Qiscus Chat SDK implementations. It offers comprehensive analytics, usage tracking, and administrative tools.

## Technology Stack

### Frontend
- jQuery - DOM manipulation
- SCSS - Styling
- Chart.js - Data visualization
- Bootstrap - UI framework

### Backend
- PHP Laravel - Web framework
- MySQL - Database
- Redis - Caching
- OAuth - Authentication

## Key Features
- Usage analytics
- Billing management
- Performance monitoring
- User statistics
- API key management
- Error tracking
- Documentation access
